Cria cursos educacionais no formato INEMA.CLUB (HTML com Tailwind, dark/light mode).

Projetos em português BR. Trabalha com o repositório iclaudex (plugin Claude Code).

Prefere planejar antes de construir e validar a estrutura com o assistente antes de gerar código.

Quer que a memória fique dentro do próprio diretório do projeto (em `.memory/`) para que se mova junto com o projeto.
