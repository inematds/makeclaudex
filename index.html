Como usar os build prompts

  1. 1 Abra o Claude Code numa pasta vazia e adicione o SPEC.md — ele é a fonte da verdade para todos os prompts.
  2. 2 Cole o conteúdo de 01_scaffold.md e aguarde o Claude construir o esqueleto.
  3. 3 Teste o que foi construído. Confirme que o plugin carrega no Claude Code antes de continuar.
  4. 4 Avance para 02_state_machine.md. Repita: construa, verifique, avance.
  5. 5 Cada prompt assume que o anterior foi concluído e testado. Nunca pule etapas.

O ponto não é copiar os prompts palavra por palavra. É ver a ordem: esqueleto antes do estado, estado antes do lifecycle, lifecycle antes das personas, personas antes do safety pass.
